🍽️ Flutter Meals App

A cross-platform Flutter application that allows users to explore a variety of meals, view detailed recipes, and apply dietary filters based on their preferences.

🚀 Features
Browse meals by categories
View detailed recipes with ingredients and steps
Mark meals as favorites
Apply dietary filters (e.g., gluten-free, vegan, vegetarian)
Clean and responsive user interface
🛠️ Tech Stack
Flutter (Dart)
Material Design UI
State management (setState / basic state handling)
